# RepoLens User Manual

## Overview

RepoLens is an AI-powered open-source repository onboarding assistant.

## Usage

### Analyze a Repository URL

1. Enter a GitHub repository URL.
2. Click "Analyze Repository".
3. Review the generated report.

### Analyze README Content

1. Paste README content.
2. Click "Analyze Repository".
3. Review the generated insights.

## Output

RepoLens generates:

* Project Summary
* Architecture Overview
* Tech Stack
* Contributor Difficulty
* Repository Health Score
* Key Components
* Contribution Roadmap
* Beginner-Friendly Tasks
