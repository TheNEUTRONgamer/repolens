# RepoLens Contributor Growth Strategy

## Objective

Build an active open-source contributor community around RepoLens.

## Week-wise Plan

### Week 1

* Improve project documentation.
* Publish contribution guidelines.
* Create issue templates for first-time contributors.

### Week 2

* Label beginner-friendly issues.
* Encourage internship peers and students to submit pull requests.

### Week 3

* Conduct repository walkthrough sessions.
* Review community pull requests and provide feedback.

### Week 4

* Recognize active contributors.
* Expand documentation and automate CI/CD workflows.

## Geographical Expansion

### Phase 1

Engineering colleges in Telangana and Andhra Pradesh.

### Phase 2

Open-source student communities across India.

### Phase 3

Global GitHub and GitLab open-source contributors.

## Community Development

* Maintain beginner-friendly documentation.
* Respond quickly to issues.
* Review pull requests promptly.
* Encourage multilingual contributions.
* Continuously improve AI analysis using community feedback.
